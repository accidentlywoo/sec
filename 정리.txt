Java Platform : 자바어플리케이션이 실행괄 환경
	JRE = JVM + API(플랫폼 별)
  - Java SE : 기본 어플리케이션, Network 어플리케이션, UI 어플리케이션 등
  - Java EE : 웹 어플리케이션, EJB 어플리케이션 등
  - Java ME : 스마트 카드 어플리케이션 등

각 플렛폼에 API가 따로 있다.

	JDK = 여러 응용프로그램(컴파일러, 인터프리터, 문서작성기) + JRE
