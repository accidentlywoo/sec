연산자
	- 산술연산자
		: +, -, *, /, %

	- 비교연산자 - true/false
		: <, >, <=, >=, ==, !=

	- 논리연산자
		: AND 연산자 &(&&), OR 연산자 |(||), Not 연산자 !
		두개짜리 연산자는 Short Surkit을 제공한다.

	- 대입연산자 - 연산 우선순위가 가장 낮다!
		: =, +=, -=, *=, /=, %=


	- 단항연산자
		: ++, --  : 다른 연산과 함께쓰면 가독성이 떨어진다!
		ex) int a = 10; b = a;
			b = ++a; // b = 11

			b = a;
			b = a++; // b = 10

	- 삼항연산자
		int a = 10;
		b = (a>10) ? 1 : 0;
